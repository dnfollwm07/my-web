### **Additional Tools**

- **Git:**
  Git is a tool for tracking changes in code, helping teams work together on the same project without overwriting each other’s work.
- **Linux:**
  Linux is an operating system commonly used on servers. It’s powerful for managing server tasks and automating processes.
- **Webpack, Rollup:**
  Webpack and Rollup are tools that package all code and resources together, so they load faster on the user’s device. This is crucial for improving website performance.
