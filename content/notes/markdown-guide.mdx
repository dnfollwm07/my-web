# Markdown 语法指南

本指南列出了当前编辑器支持的所有 Markdown 语法。您可以在编辑器中尝试这些语法，右侧会立即显示渲染效果。

## 1. 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 2. 文本样式

```markdown
*斜体* 或 _斜体_
**加粗** 或 __加粗__
~~删除线~~
==高亮==
`行内代码`
```

## 3. 列表

### 无序列表
```markdown
- 无序列表项
- 另一个列表项
```

### 有序列表
```markdown
1. 有序列表项
2. 另一个有序项
```

### 任务列表
```markdown
- [ ] 未完成任务
- [x] 已完成任务
```

### 多级列表
```markdown
- 一级
  - 二级
    - 三级
```

## 4. 引用

```markdown
> 引用文本
> 可以多行
```

### 嵌套引用
```markdown
> 一级引用
> > 二级引用
```

## 5. 代码块

```markdown
\`\`\`javascript
const code = "代码块";
\`\`\`
```

## 6. 链接

```markdown
[链接文本](https://example.com)
<https://example.com>  // 自动链接
```

## 7. 图片

```markdown
![图片描述](图片地址)
这是一张 ![行内图片](图片地址) 图片
```

## 8. 表格

### 基本表格
```markdown
| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |
```

### 表格对齐
```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 左     |   中     |    右  |
```

### 删除线表格
```markdown
| ~~删除的标题~~ | 正常标题 |
|---------------|----------|
| ~~删除的内容~~ | 正常内容 |
```

## 9. 分割线

```markdown
---
或
***
或
___
```

## 10. 数学公式

### 行内公式
```markdown
$E=mc^2$
```

### 块级公式
```markdown
$$
\sum_{i=1}^n i = \frac{n(n+1)}{2}
$$
```

## 11. 脚注

```markdown
这是一个脚注[^1]

[^1]: 这是脚注的内容
```

## 12. 行内 HTML

```markdown
<span style="color: red">红色文本</span>
```

## 13. 转义字符

```markdown
\* 显示星号而不是斜体
```

## 注意事项

1. 数学公式需要正确安装 KaTeX 相关依赖
2. 某些高级功能（如任务列表、脚注）需要 GFM 支持
3. 行内 HTML 的支持可能有限，建议使用 Markdown 原生语法
4. 所有语法都支持实时预览
5. 编辑器会自动美化渲染效果
