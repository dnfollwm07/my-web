## 注意事项

1. 数学公式需要正确安装 KaTeX 相关依赖
2. 某些高级功能（如任务列表、脚注）需要 GFM 支持
3. 行内 HTML 的支持可能有限，建议使用 Markdown 原生语法
4. 所有语法都支持实时预览
5. 编辑器会自动美化渲染效果
