## HTML5 主要功能

HTML5 構建塊是一整套技術和功能，旨在為網頁和應用程序提供更豐富的交互和功能。

## 語義性

- 可用於更**精確**地描述您的**內容**是什麼
- HTML5 提供了更具語義的標籤，使開發者能夠更準確地描述其內容的含義。例如，使用 `<article>`、`<section>`、`<header>` 和 `<footer>` 等標籤來表達不同的內容結構，這有助於**搜索引擎**和輔助技術（如**屏幕閱讀器**）更好地理解網頁內容。

### 常用語義化標籤

- `<header>` - 頁面或區塊的頂部
- `<nav>` - 導航鏈接區域
- `<main>` - 主要內容區域
- `<section>` - 文檔中的通用區段
- `<article>` - 獨立的內容單元
- `<aside>` - 側邊欄或相關內容
- `<footer>` - 頁面或區塊的底部

## 連接性

-  讓您以有效的方式**與服務器通信**。
-  HTML5 引入了新穎的 API（如 WebSockets 和 Server-Sent Events），使得客戶端與伺服器之間的通信更加高效和靈活。這允許開發者創建**即時更新**的應用，如聊天應用和**即時通知**系統。
   - Pike
     - 用於前端和後端之間實時數據傳輸和通信的工具
     - **WebSocket**: 允許雙向通信的協議
     - **HTTP/2**: 允許**多個請求**同時發送
     - **Server-Sent Events (SSE)**: 服務器可以向客戶端發送實時更新的單向通信方法
     - 前端需要: 服務器地址, 端口, , 協議, 請求URL, 標頭, 用戶識別, 會話狀態

## 離線與存儲

- 允許網頁在**客戶端****本地**存儲數據並更高效地離線操作。
- HTML5 提供了本地存儲的功能（如 Web Storage 和 IndexedDB），使得網頁能夠在客戶端本地存儲數據，並在沒有網絡連接的情況下高效運行。這使得用戶能夠在離線模式下繼續使用應用程序。

### 存儲選項

- Web Storage
  - localStorage: 將數據**永久存儲在瀏覽器中**，即使刷新頁面或關閉瀏覽器後數據依然存在
  - sessionStorage: 當前**會話中保存數據**，當瀏覽器或頁籤關閉後數據會自動刪除

- IndexDB：一種低層次的 API，用於存儲大型數據，例如 JSON 格式的數據，可以實現更複雜的數據結構和檢索功能

## 多媒體

- 使**視頻**和**音頻**在 Web 上可用，不需要其他插件
- HTML5 將視頻和音頻作為第一級公民，支持 `<video>` 和 `<audio>` 標籤，使得嵌入和播放媒體內容變得更加簡單且無需外部插件。這改善了多媒體內容的呈現和用戶體驗。

### 範例

```html
<!-- 視頻 -->
<video controls width="320" height="240">
  <source src="movie.mp4" type="video/mp4">
  您的瀏覽器不支持視頻標籤。
</video>

<!-- 音頻 -->
<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  您的瀏覽器不支持音頻標籤。
</audio>
```

## 2D3D 圖形與效果

- 支持 `<canvas>` 標籤，允許更多樣化的圖形選項
- HTML5 提供了 `<canvas>` 元素和 WebGL，允許開發者創建動態的 2D 和 3D 圖形效果，從而實現更加多樣化的視覺展示效果。

### Canvas 範例

```html
<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
  const canvas = document.getElementById("myCanvas");
  const ctx = canvas.getContext("2d");
  ctx.fillStyle = "#FF0000";
  ctx.fillRect(0, 0, 150, 75);
</script>
```

## 性能與整合

- 更大的速度優化和更好地利用計算機硬件。
- HTML5 改進了性能優化，使應用程序運行速度更快，更好地利用計算機硬件的資源，這對於高性能的網頁應用至關重要。

## 設備訪問

- 各種輸入和輸出設備的使用
- HTML5 提供了 API 來訪問各種輸入和輸出設備，如攝像頭、麥克風和位置服務，這使得開發者可以創建更具互動性和多功能性的應用。

### 常用 API

- 地理位置 API
- MediaDevices API (訪問相機、麥克風)
- 震動 API
- 電池狀態 API

## 樣式

- 讓作者編寫更複雜的 (複雜) 主題
- HTML5 使開發者能夠編寫更複雜的主題，並通過 CSS 進行更好的樣式設計和布局，使網頁外觀更加美觀和易用。
