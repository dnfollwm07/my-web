# HTML 筆記目錄

HTML (超文本標記語言) 是構建網頁的基本構建塊，定義了網頁的結構和內容。以下是 HTML 相關的主要主題：

## 基礎知識
- [HTML 基礎](basics)：DOCTYPE、多語言支持、data-屬性、語義化標籤等

## 數據存儲
- [HTML 客戶端存儲](storage)：Cookie、localStorage、sessionStorage、IndexedDB 等

## 腳本與性能
- [HTML 腳本與性能](scripting)：腳本加載、樣式表放置、漸進式渲染、事件委派等

## HTML5 特性
- [HTML5 構建塊與功能](building-blocks)：HTML5 的主要功能和新特性

## 多媒體元素
- [HTML 媒體元素](media)：圖片、Canvas、SVG 等媒體相關元素

## 相關資源

- [MDN Web Docs](https://developer.mozilla.org/zh-TW/docs/Web/HTML) - 詳細的 HTML 參考和教程
- [W3Schools](https://www.w3schools.com/html/) - 提供交互式學習體驗的教程
- [HTML5 標準](https://html.spec.whatwg.org/) - HTML 的最新標準文檔
